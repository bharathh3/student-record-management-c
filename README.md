# Student Record Management System (C)

## Description
A console-based application developed in C to manage student records using structures, pointers, and file handling.

## Features
- Add student records
- View all records
- Update and delete records
- Persistent storage using files

## Concepts Used
- C programming
- Structures and pointers
- File handling
- Dynamic memory allocation

## How to Run
gcc main.c -o student
./student
